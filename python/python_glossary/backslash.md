# Backslash

## \t = prints tab   
```python
`print("\tHello")`
	Hello
```

## \n = new line    
```python
print("Language:\nPython\nC\nSQL")
# OUTPUT: Language:
#         Python
#         C
#         SQL
```

## Quotation Marks   
```python
author = "Bill Gates"
quote = "Hello, World!"

print(f"{author} once said \"{quote}\"")
```
