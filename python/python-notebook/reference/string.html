Strings And String Methods

#string #suffix

In Python, a string is a sequence of characters that is enclosed in single quotes, double quotes, or triple quotes. Strings are immutable, which means that once created, the contents of a string cannot be modified.

Here are some basic operations that can be performed on strings in Python:

  1. Concatenation: Strings can be concatenated using the + operator. For example, "Hello" + " " + "world" would result in the string "Hello world".
  2. Repetition: Strings can be repeated using the * operator. For example, "spam" * 3 would result in the string "spamspamspam".
  3. Indexing: Individual characters in a string can be accessed using indexing. For example, "spam"[0] would return the character "s", while "spam"[3] would return the character "m".
  4. Slicing: A substring of a string can be obtained using slicing. For example, "spam"[1:3] would return the string "pa". Slicing syntax is start:stop:step, where start is the index of the first character to include, stop is the index of the first character to exclude, and step is the interval between characters to include.
  5. Formatting: Strings can be formatted using the format() method. For example, "My name is {} and I am {} years old".format("John", 30) would result in the string "My name is John and I am 30 years old".

Strings also have many built-in methods for performing common operations, such as finding substrings, replacing substrings, converting case, etc. These methods can be accessed using dot notation, such as "spam".upper() to convert the string "spam" to uppercase.

Replication

The * (asterisk) sign, when applied to a string and number (or a number and string, as it remains commutative in this position) becomes a replication operator:

string * number number * string

It replicates the string the same number of times specified by the number.

For example:

  • "James" * 3 gives "JamesJamesJames"
  • 3 * "an" gives "ananan"
  • 5 * "2" (or "2" * 5) gives "22222" (not 10!)

REMEMBER A number less than or equal to zero produces an empty string.

This simple program "draws" a rectangle, making use of an old operator (+) in a new role:

print("+" + 10 * "-" + "+")  
print(("|" + " " * 10 + "|\n") * 5, end="")  
print("+" + 10 * "-" + "+")

Note the way in which we've used the parentheses in the second line of the code.

Most commonly used string methods in Python:

  • capitalize(): Returns a copy of the string with the first character capitalized and the rest in lowercase.
  • casefold(): Returns a lowercase version of the string that is suitable for case-insensitive comparisons.
  • center(width[, fillchar]): Returns a string centered in a field of a specified width.
  • count(sub[, start[, end]]): Returns the number of occurrences of a substring within a string.
  • encode([encoding[, errors]]): Returns an encoded version of the string.
  • format(*args, **kwargs): Formats the string using positional and keyword arguments.
  • format_map(mapping): Formats the string using a dictionary or a mapping object.
  • index(sub[, start[, end]]): Returns the index of the first occurrence of a substring within a string. Raises an exception if the substring is not found.
  • len(): Returns the length of the string.
  • lower(): Returns a copy of the string with all uppercase characters converted to lowercase.
  • upper(): Returns a copy of the string with all lowercase characters converted to uppercase.
  • strip(): Returns a copy of the string with leading and trailing whitespace removed.
  • split(): Returns a list of substrings separated by a specified delimiter.
  • join(): Joins the elements of an iterable (e.g., list, tuple, set) into a string, using the string as a delimiter.
  • replace(): Returns a copy of the string with all occurrences of a specified substring replaced with another string.
  • find(): Returns the index of the first occurrence of a specified substring in the string, or -1 if not found.
  • count(): Returns the number of occurrences of a specified substring in the string.
  • startswith(): Returns True if the string starts with a specified substring, otherwise False.
  • endswith(): Returns True if the string ends with a specified substring, otherwise False.
  • isalpha(): Returns True if all characters in the string are alphabetic, otherwise False.
  • isdigit(): Returns True if all characters in the string are digits, otherwise False.
  • isalnum(): Returns True if all characters in the string are alphanumeric, otherwise False.
  • isupper(): Returns True if all characters in the string are uppercase, otherwise False.
  • islower(): Returns True if all characters in the string are lowercase, otherwise False.
